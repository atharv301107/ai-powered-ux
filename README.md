# Healthcare Admin Dashboard

Complete Admin Dashboard project using Flask + SQLite database.

## Features
- Admin login
- User list
- Add user
- Edit user
- Delete user
- Role management: Admin, Doctor, Patient, Staff
- Appointment reports
- Add/Edit/Delete appointments
- Healthcare statistics
- Dashboard charts using Chart.js
- SQLite database saves all records
